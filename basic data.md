### [DevOS EcoSystem](http://devos.76448.in/) [2022-Present]

Key features:
- **Unified Cross-Platform Architecture:** Developed a cohesive ecosystem of Arch Linux-based operating systems tailored for diverse platforms (Desktop/Laptop, Smartphone, Smartwatch, Raspberry Pi, Automobile), sharing a common core and streamlining development and user experience.
- **Integrated _DevPKG_ Package Manager:** Designed and implemented a custom package manager, DevPKG, ensuring consistent software installation, updates, and management across all DevOS platforms, simplifying the user experience and system administration.
- **Seamless Inter-Device Communication via _OctaConn_:** Pioneered an innovative, passwordless inter-device communication system, OctaConn, enabling effortless resource sharing and interaction between a user's DevOS devices within a defined range, enhancing productivity and convenience.
- **Platform-Optimized Adaptations:** Successfully customized and optimized the Arch Linux base for the unique hardware and user interface requirements of each target platform, demonstrating expertise in kernel configuration, driver integration, and UI/UX design for diverse form factors.
- **Community-Driven Open Source Initiative:** Established and led an open-source project, fostering community contributions, managing collaborative development, and promoting transparency and shared ownership in the evolution of the DevOS Ecosystem.
